17mov_dispatch_phone is a simple dispatch app for 17mov_Phone that creates dispatch calls for lb-tablet.
You can adjust the server side dispatch logic to use something else instead of lb-tablet, but only do that if you know what you’re doing.
It works, looks decent, and that’s all I need. Enjoy.
